# Sistema de Notas

Um sistema robusto de gestão escolar desenvolvido em Python, focado em **Validação de Dados** e **Persistência de Arquivos**.

## Sobre o Projeto
Este projeto foi criado como parte do meu aprendizado de Python, aplicando conceitos de Clean Code e Modularização. O sistema permite cadastrar notas, validar se os valores são numéricos e dentro da escala de 0 a 10, e gera um arquivo externo com os nomes dos alunos que atingiram a nota máxima.

## Funcionalidades
- **Modularização:** Funções específicas para entrada de dados e lógica de negócio.
- **Tratamento de Erros:** Uso de `try/except` para prevenir quebras por entradas inválidas.
- **Persistência de Dados:** Leitura e escrita em arquivo `.txt` (Mural de Honra).
- **Docstrings:** Documentação técnica interna seguindo padrões profissionais.

## Tecnologias Utilizadas
- **Python 3.x**
- **I/O de Arquivos** (Modos `r` e `a`)
- **Markdown** (Para documentação)
